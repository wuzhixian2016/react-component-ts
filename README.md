本项目用于 react typescript 学习，主要以一些 react [开源组件](https://github.com/react-component)的源码学习为主

## 项目脚本

### `npm run dev`

开发

### `npm run build`

打包

### `npm run docz:dev`

文档开发

### `npm run docz:build`

文档打包
